# Vue Jeopardy Game

## Overview
This project is a browser-based Jeopardy style trivia game built with Vue. It uses a remote trivia API to dynamically generate categories and questions, while also managing players, scoring, turns, and game flow.

## Features
- Dynamic Jeopardy board with multiple categories and questions
- Trivia questions loaded from an external API
- Multi-player score tracking
- Turn based gameplay
- Correct/incorrect answer handling
- End-of-game winner display

## Technologies Used
- Vue 3
- JavaScript
- HTML
- External API integration
- CSS

## What I Learned
- Building interactive user interfaces with Vue
- Managing application state and game logic
- Working with API calls and asynchronous data
- Structuring dynamic frontend components

## My Role
I designed and implemented the application logic, integrated the API data, and created the user interaction flow for the game.

## How to Run
1. Clone this repository
2. Open the project in your IDE
3. Install dependencies if needed
4. Start the development server
5. Open the localhost link in your browser
